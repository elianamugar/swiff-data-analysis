# SWIFF Data Analysis

A Python data analysis project exploring submission patterns from the Student World Impact Film Festival (SWIFF).

## Overview

This project analyzes a CSV dataset of SWIFF film submissions and calculates the number of submissions by country.

The script reads submission data, cleans country entries, counts frequencies, and exports the results to a text file.

## Features

- CSV data parsing
- Country frequency analysis
- Clean output generation
- Portable project structure

## Project Structure
